## 安装
1. 复制ManiaBeatmap.dll到StreamCompanion的\Plugins\net8.0目录
2. 解压mania-overlay到StreamCompanion的\Files\Web\overlays目录
## 注意
SR计算没有缓存，不过除了重霄这种要等几秒，基本都在1秒以内
